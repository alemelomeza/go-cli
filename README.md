# GO-CLI

## Installation

Download repository:
```
git clone https://github.com/alemelomeza/go-cli.git
```

Access the project folder:
```
cd go-cli
```

Build:
```
go build
```

Install:
```
go install
```

## Usage

Using `go build`:
```
./go-cli [OPTIONS]
```

Using `go install`:
```
go-cli [OPTIONS]
```

* `-lang | --lang` Languages en | es |¬†pt-br | fr (default en)
* `-name | --name` Name (default World)
* `-help | --help` Help
